# Instagram 악성 댓글 탐지기

> 댓글 알바·스팸봇이 달아놓은 악성 댓글을 자동으로 탐지하고 한 번에 삭제하는 데스크탑 앱

![Python](https://img.shields.io/badge/Python-3.11+-blue) ![Platform](https://img.shields.io/badge/Platform-Windows%20%7C%20macOS-lightgrey) ![License](https://img.shields.io/badge/License-CC%20BY--NC--ND%204.0-lightgrey)

---

## 왜 만들었나

인플루언서 게시물에는 이런 댓글이 수백 개씩 달린다.

```
성 세 라 G 컵 gⓞⓞgⓛe 검 색 ㄹㅈㄷ 미 드 지 림
>성 세 라 사 건< ġ૦૦ġĿ 검 색 미 드 지 림
[성세라 원 본] 구 글 검 색 하면 VIP 전용 방 송
```

일부러 글자를 띄우거나 유니코드 동형 문자(`ⓞ`, `ġ`, `૦`)를 섞어 Instagram 자체 필터를 우회하는 댓글 알바 패턴이다.  
이걸 하나씩 손으로 지우는 건 현실적으로 불가능하다. 이 앱은 그걸 자동으로 찾아서 묶어서 삭제한다.

**비밀번호는 이 앱에 절대 저장되지 않는다.** 브라우저에서 직접 로그인하고, 앱은 쿠키 세션만 사용한다.

## 주요 기능

| 기능 | 설명 |
|------|------|
| **단일 브라우저 흐름** | 로그인 → 게시물 선택 → 탐지 → 삭제까지 브라우저 하나로 |
| **3단계 확신도 분류** | 🔴 높음 / 🟠 중간 / 🟡 검토 탭으로 나눠서 표시 |
| **우회표기 탐지** | `gⓞⓞgⓛe`, `ġ૦૦ġĿ` 같은 유니코드 깨짐 자동 감지 |
| **캠페인 추적** | 확정 스팸에서 패턴을 학습해 변형된 단건 댓글까지 추적 |
| **키워드 직접 검색** | 필터가 놓친 댓글을 단어로 직접 찾아 삭제 |
| **전체 댓글 보기** | 수집한 모든 댓글을 실시간 필터와 함께 확인 |
| **Windows/macOS 지원** | PyInstaller + Playwright Chromium 번들로 설치 없이 실행 |

## 탐지 알고리즘

### 파이프라인

```
댓글 수집
    │
    ▼
[Step 0] 정규화
    NFKD 동형문자 접기  (ⓞ→o, ġ→g, ૦→0 등)
    NFC 재조합          (한글 자모 분리 방지)
    공백·낱자모·낱영문 제거
    우회표기 비율 측정  (obfuscation_ratio)
    │
    ▼
[Step 1] 클러스터링
    완전히 같은 텍스트 → exact 그룹
    Jaccard 유사도 ≥ 0.6 → fuzzy 병합  (Union-Find)
    │
    ▼
[Step 2] 캠페인 키워드 학습
    확정 스팸 2개+ 공통 3~4-gram 추출
    호객어(구글/검색/있음 등) 스탑리스트 제외
    → 캠페인 고유 키워드 집합
    │
    ▼
[Step 3] 신호 점수 합산 → 확신도 판정

    신호                  점수
    ─────────────────── ────
    블랙리스트 키워드      +6
    반복 3회+             +4
    강한 우회표기          +3
    반복 2회              +2
    캠페인 키워드 일치     +2
    군집 크기 확정 가중    +2
    약한 우회표기          +1
    호객어 + 캠페인 동반   +1

    ≥ 5점 → 🔴 HIGH   기본 선택
    ≥ 3점 → 🟠 MEDIUM 기본 선택
    ≥ 2점 → 🟡 LOW    기본 미선택 (검토 필요)
    < 2점 → 미탐지
```

### 오탐 방지 설계

- **최소 길이 필터**: 정규화 후 5자 미만(`ㅋㅋㅋ`, `ㅎㅎ`)은 탐지 제외
- **호객어 독립 금지**: `구글`, `검색` 단독으로는 탐지 트리거 안 됨 — 캠페인 고유 키워드 동반 필수
- **캠페인 키워드 교차 검증**: 2개 이상의 확정 스팸에 공통으로 나온 키워드만 학습
- **LOW 기본 미선택**: 애매한 탐지 결과는 체크 해제 상태로 격리 — 실수 삭제 방지

### 탐지 예시

| 입력 댓글 | 결과 | 이유 |
|-----------|------|------|
| `성 세 라 G 컵 gⓞⓞgⓛe 검 색 ㄹㅈㄷ` | 🔴 HIGH | 강한 우회표기 + 캠페인 키워드 |
| `>성 세 라 사 건< ġ૦૦ġĿ 검 색` | 🔴 HIGH | 동형문자 2개+ + 캠페인 키워드 |
| `성세라 미드지림 봤음` | 🟠 MEDIUM | 캠페인 키워드 2개+ 일치 (단건) |
| `[성세라 원 본] 구 글 검 색하면 VIP 전용 방 송` | 🟡 LOW | 호객어 + 캠페인 키워드 1개 |
| `구글에 검색해봤어요 대박` | 미탐지 | 호객어만, 캠페인 키워드 없음 |
| `성세라 너무 예뻐요` | 미탐지 | 이름만 겹침, 다른 신호 없음 |
| `ㅋㅋㅋㅋ` | 미탐지 | 정규화 후 5자 미만 |

## 설치 및 실행

### 방법 1 — 바로 실행 (Windows, 권장)

[Releases](../../releases) 페이지에서 최신 `instagram-detector-windows.zip` 다운로드 후 압축 해제 → `instagram-detector.exe` 실행.

Python, Playwright, Chromium 모두 내장되어 있어 별도 설치 불필요.

> Windows 10 / 11 (64-bit) 지원

### 방법 2 — 소스 직접 실행

**요구사항**: Python 3.11+

```bash
git clone https://github.com/hoonZeee/instagram-detector.git
cd instagram-detector

pip install -r requirements.txt

# Playwright Chromium 설치 (최초 1회)
playwright install chromium

python main.py
```

## 사용 방법

### 1단계 — 로그인

앱을 실행하면 Chromium 브라우저가 열린다. 브라우저에서 본인 Instagram 계정으로 직접 로그인한다.

> 비밀번호는 이 앱에 전달되지 않는다.  
> 로그인 세션(쿠키)만 로컬(`~/.instagram_detector/browser_profile/`)에 저장되며, 다음 실행 시 자동 복원된다.  
> **"다른 계정으로 로그인"** 버튼으로 저장된 세션을 완전 삭제할 수 있다.

### 2단계 — 게시물 선택

1. 인스타 아이디 입력 후 **"내 게시물 페이지로 이동"** 클릭
2. 브라우저에서 원하는 게시물 클릭
3. 앱의 **"이 게시물 선택"** 버튼 클릭 → URL 자동 읽기
4. 확인 카드에서 게시물 확인 후 **"이 게시물로 댓글 불러오기"**

### 3단계 — 탐지 결과 확인

| 탭 | 내용 | 기본 상태 |
|----|------|-----------|
| 🔴 높음 | 확실한 스팸 | 체크됨 |
| 🟠 중간 | 의심 스팸 | 체크됨 |
| 🟡 검토 | 애매한 항목 | **미체크** (직접 확인) |
| 🔍 검색 | 키워드로 직접 검색 | — |
| 💬 전체 댓글 | 수집된 모든 댓글 | — |

### 4단계 — 삭제

각 탭에서 `전체 선택 / 선택 해제`로 조정 후 **"선택 삭제"** 클릭. 삭제 완료 시 모든 탭 자동 갱신.

### 필터가 못 잡은 댓글 — 직접 검색

🔍 **검색 탭**에서 의심 단어 입력 시:
- **정확히 포함**: 그 단어가 들어간 댓글 전부 (기본 체크)
- **유사한 댓글 추천**: 정확 매칭과 문장이 유사한 변형 댓글 (기본 미체크)

띄어쓰기와 우회표기를 무시하고 매칭. `성 세 라` 검색 → `성세라` 포함 댓글 전부 히트.

## Windows 빌드 (개발자용)

GitHub Actions으로 자동 빌드. 태그 푸시 시 실행.

```bash
git tag v1.1.0
git push origin v1.1.0
```

Actions → `Build Windows` → Artifacts에서 `instagram-detector-windows.zip` 다운로드.

번들 구성: PyInstaller `--onedir` + Playwright Chromium (`pw-browsers/`)

## 프로젝트 구조

```
instagram-detector/
├── main.py               # 진입점 (PyInstaller frozen 환경 설정)
├── gui.py                # 앱 전체 화면 전환 관리
├── browser.py            # Playwright 세션 관리 (로그인·세션 유지)
├── detector.py           # 스팸 탐지 엔진 (정규화·클러스터링·점수화)
├── instagram.py          # Instagram API (댓글 수집·삭제)
├── playwright_worker.py  # Playwright 전용 스레드 래퍼
├── views/
│   ├── login_view.py     # 로그인 화면
│   ├── post_view.py      # 게시물 선택 (단일 브라우저 흐름)
│   └── result_view.py    # 탐지 결과 (3탭 + 검색 + 전체 댓글)
└── .github/workflows/
    └── build-windows.yml # Windows 자동 빌드
```
